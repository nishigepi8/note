## 核心理念

效率提升的本质是：

1. **减少重复输入**：别名、快捷键、自动补全
2. **减少记忆负担**：工具化常用操作
3. **减少切换成本**：快速导航、分屏操作
4. **减少等待时间**：并行、缓存、预加载
